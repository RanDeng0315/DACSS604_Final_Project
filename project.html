Final Project: Optimizing Warehousing and Logistics

This project focuses on helping small-scale entrepreneurs optimize warehousing and logistics costs using real-world data analysis.

Project Highlights

  • Analyzed Logistics Manager Index (LMI) data
  • Evaluated seasonal demand and regional differences
  • Examined transportation costs by mode
  • Integrated sustainability data for Scope 3 emissions
